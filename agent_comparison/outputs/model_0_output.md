知识点讲解:本题考查的知识点是立方根的计算，解题关键点是理解立方根的性质及其运算规则。

###板书一:

$\therefore ³√-343 = -7$

###板书二:

$\therefore -³√\left(\frac{1}{1000}\right) = -\frac{1}{10}$

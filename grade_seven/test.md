 ### 讲解一:
我们先解第一个不等式 \(4(x - 0.3) < 0.5x + 5.8\)。

首先，将括号展开：
\[ 4x - 1.2 < 0.5x + 5.8 \]

接下来，我们将含有 \(x\) 的项移到一边，常数移到另一边：
\[ 4x - 0.5x < 5.8 + 1.2 \]

合并同类项：
\[ 3.5x < 7 \]

最后，我们将不等式两边同时除以3.5：
\[ x < 2 \]

所以，第一个不等式的解集是 \(x < 2\)。

### 讲解二:
接下来我们解第二个不等式 \(3 + x \geq \frac{1}{2}x + 1\)。

首先，将含有 \(x\) 的项移到一边，常数移到另一边：
\[ 3 + x - \frac{1}{2}x \geq 1 \]

合并同类项：
\[ 3 + \frac{1}{2}x \geq 1 \]

接下来，我们将不等式两边同时减去3：
\[ \frac{1}{2}x \geq -2 \]

最后，我们将不等式两边同时乘以2：
\[ x \geq -4 \]

所以，第二个不等式的解集是 \(x \geq -4\)。

### 讲解三:
现在我们需要结合两个不等式的解集，找出它们的公共部分。

第一个不等式的解集是 \(x < 2\)，第二个不等式的解集是 \(x \geq -4\)。

所以，两个不等式同时成立的解集是：
\[ -4 \leq x < 2 \]

\[ x = -4, -3, -2, -1, 0, 1 \]

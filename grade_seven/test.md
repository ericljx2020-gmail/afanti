\[ 4x - 1.2 < 0.5x + 5.8 \]

接下来，将所有的x项移到不等号的一边，常数项移到另一边：

\[ 4x - 0.5x < 5.8 + 1.2 \]
\[ 3.5x < 7 \]

然后，我们把系数3.5变为1，所以两边同时除以3.5：

\[ x < 2 \]

这就是第一个不等式的解集。

### 讲解二:

接下来我们来看第二个不等式 \(3 + x \geq \frac{1}{2}x + 1\)。首先，我们把含有x的项移到一边，常数项移到另一边：

\[ 3 + x - \frac{1}{2}x \geq 1 \]
\[ 3 + \frac{1}{2}x \geq 1 \]

然后，将常数项移到另一边：

\[ \frac{1}{2}x \geq -2 \]

接下来，我们把系数1/2变为1，所以两边同时乘以2：

\[ x \geq -4 \]

这个不等式的解集是 \( x \geq -4 \)。

### 讲解三:

最后，我们需要找出使得两个不等式同时成立的x的整数值。我们已经知道第一个不等式的解集是 \( x < 2 \)，第二个不等式的解集是 \( x \geq -4 \)。

把两个解集结合起来，我们得到：

\[ -4 \leq x < 2 \]

在这个范围内的整数值是：-4, -3, -2, -1, 0, 1。

所以，满足这两个不等式的整数解是：-4, -3, -2, -1, 0, 1。

## 讲解：
 ./selected/slide_2320.png 
 ### 讲解一:
首先，我们解第一个不等式：4(x - 0.3) < 0.5x + 5.8。

1. 展开括号，得到：4x - 1.2 < 0.5x + 5.8。
2. 将所有含x的项移到不等式的一边，常数项移到另一边，得到：4x - 0.5x < 5.8 + 1.2。
3. 合并同类项，得到：3.5x < 7。
4. 两边同时除以3.5，得到：x < 2。

所以，第一个不等式的解集是x < 2。

### 讲解二:
接下来，我们解第二个不等式：3 + x ≥ 1/2x + 1。

1. 将所有含x的项移到不等式的一边，常数项移到另一边，得到：3 + x - 1/2x ≥ 1。
2. 合并同类项，得到：3 + 1/2x ≥ 1。
3. 将常数项移到右边，得到：1/2x ≥ 1 - 3。
4. 计算常数项，得到：1/2x ≥ -2。
5. 两边同时乘以2，得到：x ≥ -4。

所以，第二个不等式的解集是x ≥ -4。

### 讲解三:
现在，我们需要找出两个不等式解集的交集。也就是说，我们需要找到同时满足x < 2和x ≥ -4的整数值。

根据这两个条件，可取的整数值为：-4, -3, -2, -1, 0, 1。

所以，x的整数解是-4, -3, -2, -1, 0, 1。

# 原题：
有两元、五元的人民币共20张，共60元。两元和五元各几张？

# 原本板书：
### 板书一:
设两元的人民币有$x$张，五元的人民币有$y$张。

$\because$ $x + y = 20$

$\because$ $2x + 5y = 60$

### 板书二:
由$x + y = 20$，得：$y = 20 - x$

将$y = 20 - x$代入$2x + 5y = 60$，得：

$2x + 5(20 - x) = 60$

### 板书三:
$2x + 100 - 5x = 60$

$-3x + 100 = 60$

$-3x = -40$

解得：$x = \frac{40}{3}$

### 板书四:
将$x = \frac{40}{3}$代入$y = 20 - x$，得：

$y = 20 - \frac{40}{3}$

$y = \frac{60 - 40}{3}$

$y = \frac{20}{3}$

### 板书五:
$\therefore$ $x = \frac{40}{3}$，$y = \frac{20}{3}$

故答案是：两元的人民币有$\frac{40}{3}$张，五元的人民币有$\frac{20}{3}$张。

### 讲解二:

### 讲解三:

### 讲解四:
将$x = \frac{40}{3}$代入$y = 20 - x$，得到$y = 20 - \frac{40}{3}$，化简后得到$y = \frac{20}{3}$。

### 讲解五:

# 原题：
一个数的小数点向右移动两位比原数就大990，求原数是多少？

# 原本板书：
### 板书一:
设原数为$x$，
依题意得：
$x \times 100 = x + 990$

### 板书二:
$100x = x + 990$

### 板书三:
$100x - x = 990$

### 板书四:
$99x = 990$

### 板书五:
$x = \frac{990}{99}$

### 板书六:
$x = 10$

故答案是：原数是10。



# 原本解答：
### 讲解一:
首先设原数为$x$，根据题意，原数的小数点向右移动两位相当于乘以100，得到的数比原数大990。

### 讲解二:
将题意转化为方程，得到$100x = x + 990$。

### 讲解三:
将方程中的$x$移到一边，得到$100x - x = 990$。

### 讲解四:
合并同类项，得到$99x = 990$。

### 讲解五:
将方程两边同时除以99，得到$x = \frac{990}{99}$。

### 讲解六:
计算得出$x = 10$，故原数是10。

# 新解答：
### 讲解一:
我们首先设原数为$x$。根据题意，小数点向右移动两位意味着这个数乘以100。题目告诉我们，这样得到的数比原数大990。因此，我们可以列出方程：
\[ x \times 100 = x + 990 \]
这一步是为了将题目中的文字描述转化为数学表达式。

### 讲解二:
接下来，我们将方程简化。将$x \times 100$写成$100x$，这样方程变为：
\[ 100x = x + 990 \]
这一步是为了让方程更简洁，便于后续的计算。

### 讲解三:
为了将$x$的项移到方程的一边，我们从方程的两边同时减去$x$，得到：
\[ 100x - x = 990 \]
这一步是为了将所有含$x$的项集中在一边，方便求解$x$。

### 讲解四:
将左边的$x$项合并，得到：
\[ 99x = 990 \]
这一步是将同类项合并，进一步简化方程。

### 讲解五:
接下来，我们通过将方程的两边同时除以99来解$x$，得到：
\[ x = \frac{990}{99} \]
这一步是为了求出$x$的具体数值。

### 讲解六:
最后，计算出$x$的值：
\[ x = 10 \]
所以，原数是10。通过这些步骤，我们成功地找到了原数。

## 题目：
甲、乙两个人的体重差是15公斤。如果甲的体重增加12.5公斤，乙的体重减少4.8公斤，那么现在甲、乙两个人的体重差是多少？

## 板书：
知识点讲解: 本题考查的知识点是体重变化对差的影响，解题关键点是理解体重增加和减少对差值的变化。

###板书一:
原差 = 15公斤
甲增加 = 12.5公斤
乙减少 = 4.8公斤

###板书二:
因为甲增加体重，所以差增加12.5公斤
因为乙减少体重，所以差增加4.8公斤

###板书三:
新差 = 15 + 12.5 + 4.8
新差 = 32.3公斤

故答案是32.3公斤。

## 讲解：
###讲解一:
我们先知道原来的体重差是15公斤。

###讲解二:
接着，甲增加了12.5公斤，因为甲增加体重，差也会增加12.5公斤。然后，乙减少了4.8公斤，因为乙减少体重，差也会增加4.8公斤。

###讲解三:
所以，我们现在的差是原来的差15公斤，加上甲增加体重的12.5公斤，再加上乙减少体重的4.8公斤。也就是：
新差 = 15 + 12.5 + 4.8

###讲解四:
计算这个总和：
新差 = 32.3公斤

所以，现在甲和乙的体重差是32.3公斤。答案是32.3公斤。

## 题目：
张先生买了5本书和3支笔，共花费56元。如果每本书的价格是每支笔的3倍，并且书的单价比笔贵8元，那么每本书和每支笔的价格分别是多少元？

## 板书：
###板书一:
设每支笔的价格为x元，每本书的价格为y元。
依题意得:
$$y = 3x$$
$$y = x + 8$$

###板书二:
联立方程:
$$3x = x + 8$$
$$3x - x = 8$$
$$2x = 8$$
$$x = 4$$

每支笔的价格为:
$$4元$$

每本书的价格为:
$$3 \times 4 = 12元$$

故答案是每支笔的价格为4元，每本书的价格为12元。

## 讲解：
###讲解一:
首先，我们设每支笔的价格为x元，那么每本书的价格是多少呢？题目中说每本书的价格是每支笔的3倍，所以每本书的价格就是3x元。

###讲解二:
接下来，我们根据题目中的信息列一个方程。题目说张先生买了5本书和3支笔，共花费56元。那么我们可以写出这个方程：
$$5 \times 3x + 3 \times x = 56$$
也就是说，5本书的价格加上3支笔的价格等于56元。

###讲解三:
接下来我们进行化简这个方程。首先把所有的x的项合并起来：
$$15x + 3x = 56$$
$$18x = 56$$
现在我们只需要把56除以18，就可以求出每支笔的价格x：
$$x = \frac{56}{18}$$
$$x = \frac{28}{9}$$
$$x = \frac{28}{9} \approx 3.11$$

###讲解四:
每支笔的价格是3.11元（近似值），然后我们可以计算每本书的价格：
$$3 \times 3.11 = 9.33$$
所以，每本书的价格是9.33元（近似值）。

###讲解五:
通过以上步骤，我们求得每支笔的价格是3.11元，每本书的价格是9.33元。

## 题目：
计算 98×102 的值

## 板书：
知识点讲解:本题考查的知识点是乘法的简便运算，解题关键点是利用乘法的分配律进行计算。

###板书一:
$98 \times 102$

拆分计算：
$98 \times 102 = 98 \times (100 + 2)$
$= 98 \times 100 + 98 \times 2$
$= 9800 + 196$
$= 9996$

故答案是9996。

## 讲解：
###讲解一:
我们要计算 $98 \times 102$，可以通过拆分来简化计算。具体步骤如下：

1. 首先，我们把102拆分为100和2，即 $102 = 100 + 2$。
2. 然后，我们把 $98 \times 102$ 写成 $98 \times (100 + 2)$。
3. 根据乘法的分配律，我们可以把这个表达式拆成两个部分：
   $98 \times 100 + 98 \times 2$。
4. 现在分别计算这两个部分：
   - $98 \times 100 = 9800$，因为把98乘以100就是在98后面加两个零。
   - $98 \times 2 = 196$，这个可以直接计算出来。
5. 最后，把这两个部分加起来：
   $9800 + 196 = 9996$。

所以，$98 \times 102$ 的计算结果是9996。

## 题目：
找出两个不同的质数 p 和 q，使得 p+q 是质数，并且 p×q 是合数。

## 板书：
知识点讲解:本题考查的知识点是质数与合数的概念，解题关键点是找到两个不同的质数 p 和 q，使得它们的和是质数且乘积是合数。

###板书一:
设 p 和 q 为两个不同的质数且 p < q
由题意得 p + q 是质数
因为两个质数的乘积一定是合数，所以 p×q 是合数

###板书二:
设 p = 2, q = 3
则 p + q = 2 + 3 = 5 (质数)
p × q = 2 × 3 = 6 (合数)

故答案是 p = 2, q = 3

## 讲解：
###讲解一:
设 p 和 q 为两个不同的质数且 p < q。题目要求 p + q 是质数，并且 p × q 是合数。因为两个质数的乘积一定是合数，所以我们只需要找到满足条件的 p 和 q。

###讲解二:
我们可以尝试从最小的质数开始。设 p = 2, q = 3。

1. 首先验证它们的和：p + q = 2 + 3 = 5。5 是质数，所以满足条件。
2. 接着验证它们的乘积：p × q = 2 × 3 = 6。6 是合数，所以也满足条件。

因此，符合条件的两个质数是 p = 2 和 q = 3。

## 题目：
在一个笼子里有鸡和兔子共35只，它们共有94只脚。问笼子里有几只鸡，几只兔子？

## 板书：
知识点讲解: 本题考查的知识点是二元一次方程组的解法，解题关键点是设变量和列方程。

###板书一:
设鸡的只数为$x$，兔子的只数为$y$。
依题意得：
$x + y = 35$  (1)
$2x + 4y = 94$  (2)

###板书二:
将 (1) 式乘以2，得：
$2x + 2y = 70$  (3)

用 (2) 式减去 (3) 式，得：
$2x + 4y - 2x - 2y = 94 - 70$
$2y = 24$
$y = 12$

将 $y = 12$ 代入 (1) 式，得：
$x + 12 = 35$
$x = 23$

故答案是鸡有23只，兔子有12只。

## 讲解：
###讲解一:
首先，我们设鸡的只数为$x$，兔子的只数为$y$。根据题意，我们可以得到两个方程：

1. 鸡和兔子的总数是35只，所以有 $x + y = 35$。
2. 鸡有两只脚，兔子有四只脚，总共有94只脚，所以有 $2x + 4y = 94$。

###讲解二:
接下来，我们需要解这个方程组。为了方便，我们可以对第一个方程进行变形。将第一个方程乘以2，得到：

$2x + 2y = 70$。

###讲解三:
然后我们用第二个方程减去这个新的方程，得到：

$2x + 4y - 2x - 2y = 94 - 70$，简化后得到 $2y = 24$。

###讲解四:
接着，我们解这个方程，得到 $y = 12$。

###讲解五:
最后，我们将 $y = 12$ 代入第一个方程 $x + y = 35$，得到：

$x + 12 = 35$，解得 $x = 23$。

所以，笼子里有23只鸡，12只兔子。
